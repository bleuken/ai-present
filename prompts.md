# AI Image Prompts for the Educational Website

A complete prompt library for generating visuals that match the cinematic, futuristic aesthetic of the site. Every prompt is tuned for **Midjourney v6+ / DALL-E 3 / Flux / Stable Diffusion XL** and follows a unified style so the final gallery feels coherent.

## Unified Style Guide

**Aesthetic direction:** Cinematic futuristic. Semi-realistic. Vibrant volumetric lighting. Blue/purple/cyan palette dominant. Glassmorphic / holographic accents. No warm colors. No 3D-render-toon-clay look — go for refined, editorial, premium.

**Recommended global suffixes** (append to every prompt):
```
cinematic futuristic educational visual, volumetric rim lighting,
blue indigo cyan purple color palette, dark navy background,
8k, hyper-detailed, editorial photography, shallow depth of field,
premium SaaS aesthetic --ar 16:9 --style raw --s 250
```

**Negative prompt (for SD/Flux):**
```
warm colors, orange, yellow, red, cartoon, anime, clay render,
3d render plastic, low quality, blurry, watermark, text, logo,
oversaturated, noisy, grainy
```

**Aspect ratios:**
- Hero / large backgrounds: `16:9` (1920×1080) or `21:9` ultrawide
- Industry cards: `3:2` (1500×1000) or `4:3`
- Timeline items: `3:4` (portrait) or `1:1`
- Icons / small visuals: `1:1` (1024×1024)

**Color reference:**
- Indigo: `#6366f1`
- Cyan: `#22d3ee`
- Deep navy: `#05060a`
- Violet: `#a855f7`
- Pink accent: `#ec4899`

## Usage & Integration

### Recommended workflow

1. Pick the prompts that matter most (Hero, Applications, Around-Us city, Future cards are the highest visual impact).
2. Generate in your preferred tool — all prompts are tuned for Midjourney v6+, DALL-E 3, Flux Pro, and Stable Diffusion XL.
3. Save outputs to `assets/photos/` with descriptive names (`hero-brain.jpg`, `healthcare.jpg`, `city-tour.jpg`, etc.).
4. Optimize:
   ```bash
   # Convert to WebP at multiple sizes
   npx @squoosh/cli --webp '{"quality": 80}' assets/photos/hero-brain.jpg
   ```
5. I can wire them into the HTML with `<picture>` elements, lazy-loading, and retina support.

### Attribution

If you use any of these prompts, the resulting images are yours. No attribution required for the prompts themselves. If you use public stock photo services (Unsplash, Pexels, Pixabay) for any of these concepts, include attribution in the website footer.

### Recommended generators by use case

- **Photorealistic cinematic (hero, applications, future):** Midjourney v6+ or Flux Pro
- **Diagrammatic / iconographic (features, flow steps):** DALL-E 3 or Recraft
- **Abstract / data-art (benefits, timeline, finale):** Midjourney v6 with `--style raw` or Stable Diffusion XL
- **Vector / clean icons (service chips):** Recraft or use the existing inline SVGs (recommended for size)
